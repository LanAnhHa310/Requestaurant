CPSC 332 Final Project Presentation

Requestaurant

Anh Ha, Jaxon Froderberg

Tue Dec 9 2025

We are programming a restaurant review and suggestions site that will help indecisive diners choose dining venues.

Functional Requirements:

  1. Users must be able to search restaurants by price range, rating, location, dietary restrictions, atmosphere, etc., and have the database supply results that match.

  2. Users must be able to select search result-restaurants and view the restaurant's information (menu, photos, etc. ) in a popup.

  3. Users must be able to view the physical location of the selected restaurant via an extension of the google maps website, which displays the map and marker of the restaurant location within our webpage.

  4. Users must be able to view responsive interfaces that automatically adjust layout, elements, and contents to fit different mobile device screens.

  5. Users must be able to toggle between light and dark mode themes on the website

  6. Users must be able to enable a simplified view that reduces image-heavy content to improve page loading speed and minimize memory usage.

  7. Users must be able to enable a disability mode that adjusts colors, contrast, and fonts to support common forms of color blindness or impaired visibility.

  8. Users must be able to create an account by generating a username, password, and email, which will then be transferred to a database in the backend.

  9. Users must be able to input their personal preferences as extra data in their account about what kind of restaurants they want to visit, and then load that information into the search page (dietary restrictions, preferred price ranges, etc.)

  10. Users must be able to create, read, update, and delete their own reviews once they have created an account.

  11. Users must be able to submit only one unique review per registered account of each restaurant to prevent “review bombing”.

  12. Users should be able to save restaurants that they liked or would like to visit in the future in their account (bookmarking ).
